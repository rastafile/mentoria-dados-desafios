# Mentoria de Dados - Visão Geral (Overview)

## Introdução

Este ambiente foi concebido para fechar o "gap" entre o aprendizado teórico e a prática de mercado na área de engenharia e análise de dados. Muitos cursos ensinam ferramentas isoladamente, mas a nossa mentoria propõe o desenvolvimento de soluções holísticas.

## Nossa Abordagem

O projeto utiliza um cenário de e-commerce altamente realista. Não usamos datasets genéricos perfeitos; o banco de dados que você vai consumir conterá:
- Clientes inativos
- Pedidos cancelados sem pagamento
- Estornos (refunds)
- Produtos sem vendas
- Fraudes simuladas

Sua missão será extrair valor, limpar e modelar essas informações.

## Perfis Atendidos

1. **Analistas de Dados:** Focarão mais na extração SQL, criação de visões de negócios e entendimento da modelagem dimensional.
2. **Engenheiros de Dados:** Focarão em como construir a infraestrutura, otimizar os pipelines de dados, modelar o Data Warehouse e implementar testes de qualidade.

Aproveite ao máximo o ambiente! O PostgreSQL já está configurado, e os dados sintéticos lhe darão flexibilidade total para testar e errar localmente.
