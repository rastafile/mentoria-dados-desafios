# Roadmap de Evolução

Este documento guia o passo a passo da nossa mentoria. Avance de nível apenas quando sentir total segurança no anterior.

## Etapa 1: SQL Essencial e Avançado
A base de todo profissional de dados.
- **Nível Básico**: Consultas simples, filtros, ordenação e manipulação de strings/datas. (Desafios 01 a 05)
- **Nível Intermediário**: Joins complexos, agregações, Group By, subqueries correlacionadas. (Desafios 06 a 10)
- **Nível Avançado**: Window functions (Rank, Lead, Lag, Row_Number), Common Table Expressions (CTEs), otimização de queries e leitura de plano de execução. (Desafios 11 a 15)

## Etapa 2: Modelagem de Dados
Como os dados devem ser estruturados para análise ou armazenamento escalável.
- **Nível Básico**: Entendimento das tabelas transacionais atuais, Normalização (1FN, 2FN, 3FN).
- **Nível Intermediário**: Desenho de modelo Star Schema (Fato e Dimensão), criação de views materializadas.
- **Nível Avançado**: Modelagem Data Vault ou estratégias de Slowly Changing Dimensions (SCD Tipo 2).

## Etapa 3: Pipelines de Dados
A movimentação dos dados de um ponto a outro de forma orquestrada e resiliente.
- **Nível Básico**: Scripts Python com Pandas/SQLAlchemy para extração full.
- **Nível Intermediário**: Carga incremental usando Python, particionamento de dados.
- **Nível Avançado**: Introdução a orquestradores (Airflow, Mage ou Dagster), testes de qualidade (Great Expectations) e CI/CD.
